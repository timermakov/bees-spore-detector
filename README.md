# Bees Spore Counter

Автоматический подсчёт спор на микроскопических изображениях камеры Горяева.

## Структура проекта

- `data/` — изображения и метаданные
- `res/` — результаты анализа (Markdown)
- `bees/` — основной код

## Запуск

1. Установите зависимости:
   ```
   pip install -r requirements.txt
   ```
2. Запустите анализ:
   ```
   python -m bees.main
   ```
